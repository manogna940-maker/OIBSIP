# OIBSIP - Android App Development Internship

This repository contains the Android App Development internship tasks completed as part of the **Oasis Infobyte Summer Internship Program (OIBSIP)**.

A simple Android application that converts values between different measurement units.

- Easy-to-use interface
- Input validation
- Instant conversion

### Technologies
